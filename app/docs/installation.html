Installation

Get started with Reactolith UI in your project.

Install via npm

Reactolith UI is distributed as an npm package. Install it alongside its peer dependencies:

npm install @reactolith/ui reactolith react react-dom

Prerequisites

  • Node.js 18+
  • A project with Tailwind CSS v4 configured
  • Reactolith installed in your project

Project Setup

If you're starting a new project, set up the basic structure:

# 1. Create a new project
mkdir my-app && cd my-app
npm init -y

# 2. Install dependencies
npm install @reactolith/ui reactolith react react-dom
npm install -D tailwindcss @tailwindcss/vite vite @vitejs/plugin-react

CSS Setup

Make sure your main CSS file imports Tailwind and the component styles:

@import "tailwindcss";
@import "tw-animate-css";

Available Components

Browse all 54 component groups in the sidebar to see what's available. All components are included in the package and resolved automatically by the loader.
