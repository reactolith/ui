Input Group

shadcn/ui docs

Groups input with addons, buttons, or text for combined input fields.

Preview

https://

Usage

<ui-input-group>
  <ui-input-group-text>https://</ui-input-group-text>
  <ui-input-group-input placeholder="example.com"></ui-input-group-input>
</ui-input-group>

Sub-components

<ui-input-group> <ui-input-group-input> <ui-input-group-addon> <ui-input-group-button> <ui-input-group-text> <ui-input-group-textarea>

All components also accept standard HTML attributes (class, id, style, data-*, aria-*, etc.) which are passed through to the underlying element.
