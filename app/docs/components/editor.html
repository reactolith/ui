Plate.js docs

Editor

A rich text editor powered by Plate.js. Configurable toolbar, plugin presets, and native form integration via hidden inputs.

Preview

Usage

<!-- Standard editor with form sync -->
<ui-editor name="content" format="html" toolbar="standard"></ui-editor>

<!-- Minimal toolbar -->
<ui-editor name="body" toolbar="minimal" format="markdown"></ui-editor>

<!-- Full-featured editor -->
<ui-editor name="content" toolbar="full" plugins="full" min-height="300px"></ui-editor>

<!-- Custom toolbar features -->
<ui-editor name="content" toolbar-features="bold,italic,link,heading"></ui-editor>

<!-- Read-only display -->
<ui-editor value="&lt;p&gt;Hello world&lt;/p&gt;" format="html" read-only></ui-editor>

<!-- No toolbar -->
<ui-editor name="notes" toolbar="none" placeholder="Quick note..."></ui-editor>

Form Integration

When the name attribute is set, the editor renders a hidden <input> that syncs the editor content (debounced at 300ms). The content is serialized in the format specified by format.

<form action="/api/save" method="POST">
  <!-- Editor content is submitted as the "content" field -->
  <ui-editor name="content" format="html"></ui-editor>

  <button type="submit">Save</button>
</form>

<!-- JSON format (lossless, recommended for storage) -->
<ui-editor name="content" format="json"></ui-editor>

<!-- Markdown format -->
<ui-editor name="content" format="markdown"></ui-editor>

noneNo toolbar rendered

Plugin Presets

PresetPlugins included
minimalBasic Marks, Heading, List, Link, Markdown
standardBasic Marks, Basic Blocks, Highlight, List, Link, Image, Table, Code Block, Markdown, Autoformat
fullAll standard plugins + Subscript, Superscript, Callout, Toggle, Indent

Extending with TypeScript

React/TypeScript developers can import the building blocks to create custom editor variants:

import { Plate, usePlateEditor } from "platejs/react";
import {
  getPluginsForPreset,
  getEditorComponents,
  EditorToolbar,
  useEditorFormSync,
  resolveToolbarFeatures,
} from "@reactolith/ui";

// Custom editor with additional plugins
function CustomEditor() {
  const editor = usePlateEditor({
    plugins: [
      ...getPluginsForPreset("standard"),
      MyCustomPlugin,
    ],
    override: { components: getEditorComponents() },
  });

  return (
    <Plate editor={editor}>
      <EditorToolbar features={resolveToolbarFeatures("standard")} />
      <PlateContent />
    </Plate>
  );
}

Props

<ui-editor>

PropTypeDefaultDescription
valuestring | Value—Initial content. Can be an HTML string, JSON string, Markdown string, or Plate Value array depending on format.
format"html" | "json" | "markdown""html"Content format for both initial value parsing and form sync serialization.
namestring—Hidden input name for form submission. When set, syncs editor content to a hidden <input>.
formstring—Form ID to associate with (like the native form attribute on inputs).
toolbar"full" | "standard" | "minimal" | "none""standard"Toolbar preset. Determines which formatting buttons are shown.
toolbar-featuresstring—Comma-separated toolbar features. Overrides preset. E.g. "bold,italic,link,heading".
plugins"full" | "standard" | "minimal""standard"Plugin preset. Determines which editing features are available.
placeholderstring"Start writing..."Placeholder text shown when the editor is empty.
read-onlybooleanfalseRead-only mode. Hides toolbar and disables editing.
min-heightstring"150px"Minimum editor height (CSS value).
max-heightstring—Maximum editor height (CSS value). Enables scroll when content exceeds this height.
