Plate.js docs

Editor

A rich text editor powered by Plate.js. Configurable toolbar, plugin presets, and native form integration via hidden inputs.

Inline Editor

Only inline text formatting (bold, italic, underline, etc.) — no block types. Ideal for single-line fields like titles or short descriptions.

<ui-editor toolbar="inline" plugins="inline" placeholder="Format text inline..."></ui-editor>

Standard Editor

Block-level editing with headings, lists, quotes, code blocks. Includes a slash menu (type / to insert blocks). The default preset.

<ui-editor name="content" format="html"></ui-editor>
<!-- or explicitly: toolbar="standard" plugins="standard" -->

Full Editor

All available features including images, tables, callouts, toggles, code highlighting, superscript/subscript, and more.

<ui-editor toolbar="full" plugins="full" min-height="250px"></ui-editor>

No Toolbar (Slash-only)

Clean writing experience without a toolbar. All block types are still available via the slash menu.

<ui-editor toolbar="none" placeholder="Type / to insert blocks..."></ui-editor>

Usage

<!-- Standard editor with form sync -->
<ui-editor name="content" format="html"></ui-editor>

<!-- Inline-only (no blocks) -->
<ui-editor toolbar="inline" plugins="inline" name="title"></ui-editor>

<!-- Full-featured editor -->
<ui-editor toolbar="full" plugins="full" name="content" min-height="300px"></ui-editor>

<!-- Custom toolbar features -->
<ui-editor toolbar-features="bold,italic,link,heading" name="content"></ui-editor>

<!-- Read-only display -->
<ui-editor value="&lt;p&gt;Hello world&lt;/p&gt;" format="html" read-only></ui-editor>

Form Integration

When the name attribute is set, the editor renders a hidden <input> that syncs the editor content (debounced at 300ms). The content is serialized in the format specified by format.

<form action="/api/save" method="POST">
  <!-- Editor content is submitted as the "content" field -->
  <ui-editor name="content" format="html"></ui-editor>

  <button type="submit">Save</button>
</form>

<!-- JSON format (lossless, recommended for storage) -->
<ui-editor name="content" format="json"></ui-editor>

<!-- Markdown format -->
<ui-editor name="content" format="markdown"></ui-editor>

Slash Menu

In standard and full presets, typing / opens a Notion-like block menu. Navigate with arrow keys, select with Enter, filter by typing. Available blocks:

Text Heading 1-3 Bullet List Numbered List Quote Code Divider Table Image

The slash menu is disabled for the inline preset since it only supports inline formatting.

Extending with TypeScript

React/TypeScript developers can import the building blocks to create custom editor variants:

import { Plate, usePlateEditor } from "platejs/react";
import {
  getPluginsForPreset,
  getEditorComponents,
  EditorToolbar,
  useEditorFormSync,
  resolveToolbarFeatures,
} from "@reactolith/ui";

// Custom editor with additional plugins
function CustomEditor() {
  const editor = usePlateEditor({
    plugins: [
      ...getPluginsForPreset("standard"),
      MyCustomPlugin,
    ],
    override: { components: getEditorComponents() },
  });

  return (
    <Plate editor={editor}>
      <EditorToolbar features={resolveToolbarFeatures("standard")} />
      <PlateContent />
    </Plate>
  );
}
